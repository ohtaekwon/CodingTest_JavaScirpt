### 15.4.1 예제1

**1.1) 입력코드 입력 1** ✍

```javascript
            function solution(arr){         
                let answer=[];
                answer.push(arr[0]);
                for(let i=1; i<arr.length; i++){
                    if(arr[i]>arr[i-1]) answer.push(arr[i]);
                }
                return answer;
            }

            let arr=[7, 3, 9, 5, 6, 12];
            console.log(solution(arr));
```
